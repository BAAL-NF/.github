# ðŸ§¬ BAAL-NF: Bayesian Analysis of ALlelic imbalance (BAAL) with NextFlow (NF)

Welcome to **BAAL-NF**, a collaborative bioinformatics organization dedicated to developing robust, reproducible, Nextflow-based workflows and computational tools for analyzing transcription factor binding, allele-specific binding (ASB), and downstream functional genomics analyses.

Below you will find our main repositories along with their persistent **Zenodo Concept DOIs**, providing stable citations for all archived software releases.

## ðŸ§ª Mission Statement

BAAL-NF aims to:

- Enable portable, scalable and reproducible analyses via Nextflow and containerization
